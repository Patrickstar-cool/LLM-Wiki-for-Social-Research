## 概述

质性研究（Qualitative Research）不是"不够量化的研究"——
它是一种**不同的求知方式**。如果说定量研究问"多少""多大""什么原因"——
质性研究问**"什么意思""经历是什么感觉""过程是怎么展开的"**。

> 定量方法把社会世界变成数字以发现规律。
> 质性方法深入社会世界以**理解意义**。

## 哲学基础

质性研究的认识论基础通常在[[诠释主义]]传统：
- 社会现实不是"客观存在、等待被发现"的——它在人的**意义赋予**过程中被建构
- 韦伯的 **Verstehen**（理解）：社会学不只是"说明"行为的原因——要"理解"行动者赋予自己行为的主观意义
- 你不能"客观地"测量"爱"——但你可以在访谈中听到一个人讲述"爱"的经历、在观察中看到"爱"如何被"做出来"

## 核心方法

### 1. 深度访谈（In-depth Interview）

不只是"问问题"——是**进入另一个人的意义世界**。

| 类型 | 特点 |
|------|------|
| **结构化** | 固定的问题、固定的顺序——像口头的问卷 |
| **半结构化** | 有主题指南，但问题和顺序灵活调整——最常用 |
| **非结构化** | 只有大方向——"跟我说说你的童年"——像对话 |

**访谈不是"提取信息"——是"共同建构意义"。** 你的提问方式、回应、沉默——
都在塑造受访者说什么、怎么说。一个好的质性研究者对这种**互动本身**保持反思。

**生活史访谈**（life history interview）：让一个人讲完一生的故事——从中看到
"个人"如何被"时代"塑造，也看到个人如何在时代中做出选择。

### 2. 参与观察（Participant Observation）

研究者的身体和感官——**进入被研究者的世界**。

- **完全参与者**：隐藏研究者身份——像间谍——获得最"真实"的数据但面临伦理问题
- **参与即观察**：公开身份，同时参与日常活动——像芝加哥学派的民族志
- **观察即参与**：以观察为主，参与为辅

这需要社会学想象力+个人勇气的结合：你必须在"融入"和"保持分析距离"之间持续平衡。
**过度融入失去分析能力，过度距离失去理解深度。**

### 3. 焦点小组（Focus Group）

默顿在 1940 年代发明的——6-10 人的小组讨论，由一位主持人引导。

- 优势：参与者之间的**互动**本身产生数据——一个人的话引发另一个人的回应、
  反驳、补充——这种动态是单人访谈无法获得的
- 劣势：强势参与者主导讨论、群体规范抑制异见

### 4. 文本/话语分析

不访问活人——分析**已经存在的文本**：政策文件、媒体报道、日记、信件、社交媒体帖子。

- 不追问"作者的真实意图是什么"
- 追问"这个文本在**做什么**？"——它在建构什么"真相"？它在排除什么声音？

## 抽样的不同逻辑

质性研究不追求"统计代表性"——它追求**"理论饱和度"**：

| 概念 | 说明 |
|------|------|
| **目的性抽样** | 根据研究问题有目的地选择"信息丰富"的个案 |
| **最大差异抽样** | 选择差异最大的个案——覆盖现象的"全光谱" |
| **滚雪球抽样** | 通过受访者介绍下一个受访者——适用于隐蔽群体 |
| **理论抽样** | 边收集数据边分析——分析结果指引下一步收集谁的数据 |

**样本大小**：不是"越多越好"——是**"直到没有新的洞见出现为止"**。
当第 11、12、13 个受访者都在重复已知的模式——你达到了"理论饱和"。

## 数据分析

质性分析不是"在数据收集完成后才开始的"——**分析和收集是同时进行的。**

### 编码（Coding）

将原始数据（访谈文字稿、田野笔记）分解为**主题标签（编码）**：

- **开放编码**：第一次阅读——标记任何可能有趣的内容——"这里好像是在说'羞耻'"
- **轴心编码**：将开放编码归类——"'羞耻''丢脸''抬不起头'都指向一个更抽象的概念？"
- **选择性编码**：找到一个"核心范畴"——整个故事围绕它展开

### 扎根理论（Grounded Theory）

格拉泽和斯特劳斯（1967）提出：**理论应该从数据中"扎根"生长出来——不是先有理论再找数据。**

这是质性方法论史上最著名的争论：研究者到底能不能"不带理论"进入田野？
后来的共识是：不可能完全"空着脑袋"——但你必须**让数据有"反驳你的预期"的机会**。

### 叙事分析

不是将访谈内容"编码"为变量——而是**保持故事的完整性**：
- 这个人是怎么组织自己人生故事的？高潮在哪？转折在哪？
- 叙事不是"反映"生活——叙事在**塑造**生活——你说什么样的故事，你就是什么样的人

## 质性研究的质量标准

不能用量化的"信度""效度"标准来评判质性研究——它需要自己的质量框架：

| 标准 | 说明 |
|------|------|
| **可信度**（credibility） | 研究结果对"内部人"来说是否可信？——让受访者审阅你的分析（成员检验） |
| **可转移性**（transferability） | 你的"深描"是否足够丰富，让读者能判断"这适用于我的情境吗？" |
| **可靠性**（dependability） | 研究过程是否透明？——留下"审计轨迹" |
| **可确认性**（confirmability） | 你的发现是否来自数据——而不是你的偏见？——保留原始数据供他人审查 |
