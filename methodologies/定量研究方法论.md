## 概述

定量研究（Quantitative Research）是社会科学中将社会现象转化为**数字**、
通过统计分析来检验假设的方法体系。它的核心承诺是：

> 社会世界尽管复杂，但可以通过**测量、计算和统计推断**来发现其中的规律。

定量方法的哲学根基在[[实证主义]]——社会事实可以像自然事实一样被客观测量。
但今天的定量研究者大多不是朴素的实证主义者——他们知道"客观"是理想而非现实。

## 核心逻辑

### 从理论到数据再到理论

```
理论 → 假设 → 操作化 → 数据收集 → 统计分析 → 回归理论
```

1. **理论**：提出变量之间的关系——"教育水平影响收入"
2. **假设**：将理论陈述转化为可检验的预测——"受教育年限每增加一年，年收入增加 X%"
3. **操作化**：将抽象概念转化为可测量的指标——"教育水平"=受教育年限？学位等级？
4. **数据收集**：通过问卷调查、行政数据、实验获取数据
5. **统计分析**：用数学方法检验假设
6. **回到理论**：结果支持理论吗？需要修正吗？

### 变量语言

定量方法的核心操作是把世界分解为**变量**：

- **因变量**（dependent variable）：要被解释的现象——收入、投票行为、健康状况
- **自变量**（independent variable）：用来解释的因素——教育、年龄、性别
- **控制变量**：在分析"教育→收入"时，剔除年龄、性别、地域的干扰效果

> 社会学家关心的不是张三和李四的个人差异——
> 而是**在控制了其他因素后，某个因素是否系统地影响了结果？**

### 相关与因果

定量方法中最核心、也最容易被误解的区分：

- **相关**：A 和 B 一起变化——教育高的人收入也高
- **因果**：A 的变化**导致**了 B 的变化——更多的教育**使**人获得更高收入

> "冰淇淋销量和溺水死亡正相关"——不是因为冰淇淋引起溺水，
> 而是因为"夏天"同时引起了更多人吃冰淇淋和更多人去游泳。

建立因果关系需要：相关关系 + 时间先后（因在前，果在后）+ **排除其他解释**。

## 主要方法

### 1. 问卷调查（Survey Research）

最广泛使用的定量方法。

- **抽样**：从总体中选取代表性的子集——不可能问每一个人
  - 随机抽样：每个人被选中的概率相等——理想但昂贵
  - 分层抽样：将总体分为亚群，在每个亚群内随机抽样
- **问卷设计**：问题的措辞、顺序、选项设置——每一个细节都影响回答
- **信度**（reliability）：重复测量能得到一致结果吗？
- **效度**（validity）：你测的真是你想测的吗？——"我问他收入"和"他真的告诉我真实收入"是两回事

### 2. 实验法（Experimental Method）

唯一能严格建立因果的方法：**随机分配**。

- 将研究对象随机分为实验组和控制组
- 实验组接受"处理"（treatment），控制组不接受
- 如果两组之间有统计显著差异——且随机分配成功→ 处理是原因

但社会科学的实验面临严重的**外部效度**问题：
实验室里的行为能推广到真实世界吗？

**自然实验**（natural experiment）：某种外生事件（政策变化、自然灾害）偶然地
把人分成了"处理组"和"控制组"——研究者利用这种自然的随机性来推断因果。

### 3. 次级数据分析

使用已有的数据——普查数据、行政记录、既往调查数据——来回答新问题。

- 成本低——不需要自己收集
- 但受到数据的限制：变量不是"为你"设计的

## 关键概念

### 统计显著性与 p 值

- **p 值**：如果"实际上没有效果"——观察到当前效果（或更极端）的概率
- 传统上 p < 0.05 被视为"显著"
- **重大危机**：2010 年代后，"p 值操控"（p-hacking）、"发表偏倚"等问题被大量揭露——**"可重复性危机"**冲击了整个社会科学
- 这不是定量方法本身的问题——是**怎么使用和解释**定量方法的问题

### 多元回归

统计分析的主力工具：
> 在"控制"其他变量不变的情况下，X 对 Y 的"独立"效应是什么？

简单说：把所有可能干扰的因素放进方程，然后看你要研究的那个因素——
在其他人"都一样"的情况下——还剩下多少解释力。

局限性：**回归只能控制你测量了的变量**——你没测到的（遗漏变量偏误）是定量研究永恒的阿喀琉斯之踵。

## 定量 vs 质性：永恒的方法论之争

| | 定量 | 质性 |
|---|---|---|
| **目标** | 检验假设、发现因果 | 理解意义、探索过程 |
| **逻辑** | 演绎（理论→假设→检验） | 归纳（数据→模式→理论） |
| **样本** | 大、随机、追求代表性 | 小、有目的、追求深度 |
| **收集** | 问卷、实验、统计数据 | 访谈、观察、文本 |
| **产出** | 数字、系数、p 值 | 描述、叙事、理论 |
| **优势** | 可推广、可复制 | 深度、灵活性 |
| **劣势** | 表面、失去上下文 | 主观、难以推广 |

今日主流：**混合方法**（mixed methods）——结合两者优势。不把它们当做"对立"，
而是"互补"。
