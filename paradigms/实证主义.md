## 核心主张

**社会现象可以像自然现象一样被客观地观察、测量和解释。**
实证主义主张社会科学应该模仿自然科学的方法论——通过经验观察、量化测量、
假设检验来发现社会规律。

一句话：**"社会世界和自然世界遵循同样的科学逻辑。"**

## 哲学渊源

- **奥古斯特·孔德（Auguste Comte，1798-1857）**：实证主义之父，首次提出
  "社会学"一词，认为人类知识经历神学→形而上学→实证三个阶段
- **埃米尔·涂尔干（Émile Durkheim，1858-1917）**：将实证方法系统应用于社会学，
  代表作《自杀论》通过统计数据证明——看似最私人的行为也有社会规律
- **逻辑实证主义（维也纳学派，1920s）**：强调"可验证性原则"——
  一个命题有意义，当且仅当它可被经验验证

## 核心原则

| 原则 | 说明 |
|------|------|
| 客观性 | 研究者应价值中立，排除个人偏见 |
| 经验主义 | 知识必须基于可观察、可测量的现象 |
| 因果解释 | 目标是发现变量间的因果关系，而非仅仅理解意义 |
| 可重复性 | 研究结果应能被其他研究者复现 |
| 量化优先 | 偏好定量数据（数字、统计）而非定性描述 |

## 对本学科的影响

实证主义深刻塑造了现代社会科学的研究实践：
- **研究方法**：问卷调查、实验法、统计分析成为标配
- **学术发表**：定量研究长期占据顶刊主流，"显著性"（p-value）成为通行证
- **政策制定**：基于证据的政策（evidence-based policy）依赖实证数据
- **学科地位**：帮助社会学、政治学等获得了"科学"的学术合法性

## 批评与局限

1. **忽视主观意义**：人不是原子——社会行动具有意义和意图，纯客观测量可能
   遗漏最重要的东西（[[诠释主义]] 的核心反驳）
2. **价值中立的幻象**：选择什么研究、用什么方法、怎么解读数据——每一步都
   隐含价值判断
3. **过度简化**：将复杂的社会现象简化为几个变量，可能丢失关键脉络
4. **理论先于观察**：观察本身受理论预设的影响（"观察负载理论"）
5. **忽略权力和结构**：实证主义倾向于描述现状而非批判，可能成为
   维护既有权力结构的工具（[[批判理论]] 的批评）

## 当代发展

**后实证主义（Post-Positivism）** 修正了经典实证主义的教条：
- 承认客观知识是理想而非现实，研究者只能"逼近"真相
- 接受理论对观察的引导作用
- 接纳定性方法或混合方法的补充价值
- 代表人物：托马斯·库恩（《科学革命的结构》）
