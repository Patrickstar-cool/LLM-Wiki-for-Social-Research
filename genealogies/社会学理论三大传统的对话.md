## 概述

马克思、涂尔干、韦伯——社会学的三大奠基人各自开启了不同的理论传统。
这三种传统不是平行存在的，它们在 150 年的对话、对抗与融合中共同塑造了社会学。

这里不是三篇独立的人物介绍，而是**三条河流的交汇、分流与合流**。

---

## 三条河流的源头

### 河流一：马克思 → 冲突传统

**核心问题**：「为什么有人富有人穷？历史向何处去？」

马克思把社会学分析的焦点放在**物质生产和社会阶级**上：
- 生产方式决定社会结构（"经济基础决定上层建筑"）
- 阶级斗争是历史的动力
- 资本主义的本质是**剥削**——工人创造的剩余价值被资本家占有
- 资本主义必然被更高级的社会形态取代

**这条河的流向**：
马克思 → 列宁（革命实践） → 葛兰西（文化霸权） → 法兰克福学派（批判理论） → 阿尔都塞（结构马克思主义） → 沃勒斯坦（世界体系） → 当代新马克思主义

### 河流二：涂尔干 → 功能/秩序传统

**核心问题**：「社会是怎么维系在一起的？秩序从何而来？」

涂尔干把焦点放在**社会整合**上：
- 社会是一个"自成一类"的实在——"社会事实"（social facts）
- 前现代社会靠"机械团结"（相似性），现代社会靠"有机团结"（分工互补）
- 犯罪不是病态——惩罚犯罪恰恰是在强化集体道德
- 《自杀论》：连"想死"这种最私人的行为都有社会规律

**这条河的流向**：
涂尔干 → 马林诺夫斯基/拉德克利夫-布朗（功能主义人类学） → 帕森斯（宏大功能论/AGIL） → 默顿（中层理论/修正功能论） → 亚历山大（新功能主义）

### 河流三：韦伯 → 理解/互动传统

**核心问题**：「现代性的代价是什么？人的行动有意义吗？」

韦伯把焦点放在**意义和理性化**上：
- 社会学必须"理解"（Verstehen）行动者的主观意义
- 资本主义在西方的兴起与新教伦理有选择性亲和
- 理性化是把双刃剑——带来效率，也带来"铁笼"
- 三种权威类型：传统型、魅力型、法理型

**这条河的流向**：
韦伯 → 舒茨（现象学社会学） → 米德/布鲁默（符号互动论） → 戈夫曼（拟剧论） → 加芬克尔（常人方法论） → 伯格&卢克曼（社会建构论）

## 合流与对话

三条河不是隔离的。以下是几次关键的"交汇"：

### 对话一：功能论 vs 冲突论（1950s-1970s）

帕森斯的功能论描绘了一个和谐、整合的社会图景。冲突论者质问：
**"谁的社会秩序？以谁的代价？"**

- 达伦多夫：社会不只是"共识"，也是"强制"——每个社会结构里都有统治和被统治
- 科塞：冲突本身也有"功能"——外部冲突增强内部团结，安全阀机制释放张力
- 米尔斯《权力精英》："帕森斯的宏大理论既不能解释越战，也不能解释贫富分化"

这场辩论的结果：**功能论退潮，冲突论成为新主流**。但后来的理论家意识到：
社会既需要整合（功能论的洞见），也存在支配（冲突论的洞见）——真正的问题是如何将两者纳入一个框架。

### 对话二：结构与行动（1970s-1980s）

- 结构论者（功能论、冲突论、结构主义）强调**结构决定行动**——人在结构中身不由己
- 互动论者（符号互动论、常人方法论、现象学）强调**行动建构结构**——结构是人做出来的

**超越二元对立的关键尝试**：
- 吉登斯"结构化理论"：结构不是外在的牢笼——我们在运用结构的同时也在再生产/改变结构
- 布尔迪厄"惯习"概念：惯习是"被结构化的结构"和"起结构化作用的结构"——它既来自社会位置，又主动生成实践

### 对话三：现代 vs 后现代（1980s-1990s）

三大传统都共享某种"宏大叙事"——历史有方向、理性有力量、解放有可能。
后现代主义把这整个框架都质问了一遍：

- 福柯：知识就是权力——社会学声称"客观认识社会"，但这本身就在生产权力
- 利奥塔："大叙事"死了——马克思的历史唯物主义、韦伯的理性化、功能论的"现代社会"都是故事
- 鲍德里亚：消费社会里符号比实物更重要——"阶级"还是有效的分析工具吗？

这场对话的结果不是"后现代赢了"，而是迫使三大传统进行自我反思：
- 新马克思主义吸收了福柯的"话语/权力"分析
- 批判理论中的"承认"转向（霍耐特）——冲突不止是物质分配，更是尊严的争夺
- 女性主义、后殖民、酷儿理论提供了传统社会学长期忽略的视角

## 当代：新的综合？

进入 21 世纪，三条河的边界越来越模糊：

- **批判实在论**（罗伊·巴斯卡）：社会科学可以既是科学的（实证面向），又承认结构不可直接观察（批判面向），还需要理解意义（诠释面向）
- **分析社会学**（赫斯特罗姆等）：用清晰的因果机制重建社会学解释——回到默顿的"中层理论"
- **关系社会学**（埃米尔巴耶、克罗斯利）：社会的本质是关系——超越"结构 vs 行动"的古老争论
- **计算 + 传统**：大数据和模拟不是取代理论，而是让三大传统的问题可以以新方法探索

### 这条河往哪流？

当下的方向可能是：**既要有理论深度，又要能回应真实世界的问题**——
不平等、气候变化、数字化、全球移民、民主危机。三大传统没有"过时"——
它们提出的问题（秩序、冲突、意义）仍然是核心问题。
变的是回答这些问题的方法和框架。

## 三大传统一览

| | 马克思传统 | 涂尔干传统 | 韦伯传统 |
|---|---|---|---|
| **核心问题** | 不平等与变革 | 整合与秩序 | 意义与理性化 |
| **分析单位** | 阶级、生产方式 | 社会事实、制度 | 社会行动、主观意义 |
| **方法** | 历史唯物论 | 实证/统计 | 理解/理想类型 |
| **社会图像** | 矛盾与革命 | 有机整体 | 多元因果网络 |
| **现代性诊断** | 异化 | 失范 | 铁笼/祛魅 |
| **当代遗产** | 批判理论、后殖民 | 功能论、网络分析 | 互动论、建构论 |
