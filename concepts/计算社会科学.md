## 定义

计算社会科学（Computational Social Science, CSS）是**以理论驱动的方式
运用计算方法来分析、建模和模拟社会系统**的交叉学科。
它位于计算机科学、数据科学与社会科学的交汇处，
试图用新工具回答社会科学的经典问题。

CSS 学会（CSSSA）的定义：
> "计算社会科学是对社会系统的理论驱动的计算分析、建模和模拟——
> 包括大规模社会数据分析、通过模拟探索社会动态的内在机制，
> 以及将洞见与社会理论连接。"

## 为什么需要计算社会科学？

传统社会科学面临三个瓶颈，CSS 恰好回应了它们：

| 传统瓶颈 | CSS 的回应 |
|----------|-----------|
| 数据有限（问卷调查几百人） | 大数据（社交媒体、手机信令、行政记录——亿级数据点） |
| 方法受限（统计回归只能处理少数变量之间的线性关系） | 机器学习、网络分析、自然语言处理——处理高维非线性关系 |
| 难以实验（社会不能随意"做实验"） | 基于代理的建模（ABM）——在虚拟世界中检验理论 |

## 核心方法

### 1. 基于代理的建模（Agent-Based Modeling, ABM）

在计算机中创建虚拟"代理"（agents）——每个代理有简单的行为规则，
研究者观察大量代理互动如何"涌现"出宏观社会模式。
例如：种族隔离如何从"只要周围有 30% 同类"的个体偏好中涌现？

2026 年前沿：AI 辅助模型生成与评估、ABM 与机器学习的混合方法。

### 2. 社会网络分析（Social Network Analysis）

将社会关系抽象为"节点"和"连线"——
谁认识谁、谁影响谁、信息如何传播。
从"小世界网络"到"无标度网络"，
网络分析揭示了社会结构的深层数学规律。

### 3. 大数据与机器学习

- **计算文本分析**（NLP）：从海量文本中提取主题、情感、意识形态
- **大语言模型（LLM）在 CSS 中的应用**：用 LLM 编码社会文本、
  模拟社会互动、生成研究假设（CSS 2026 的热点方向）
- **地理空间分析**：将社会数据映射到物理空间

### 4. 在线实验与社会计算

利用互联网平台进行大规模社会实验——
Facebook 的"情绪传染实验"（2014）是争议性的经典案例。

## 学术生态

### 核心组织

- **CSSSA**（计算社会科学学会美洲分会）：主办 CSS 年度会议
- **IC2S2**（国际计算社会科学会议）：全球性学术聚会
- 2026 年 CSS 年会将于 **10 月 29 日-11 月 1 日**在新墨西哥州圣塔菲举行，
  主题演讲人 Ada Palmer 和 Marco Janssen

### 主要期刊

- *Journal of Computational Social Science*
- *EPJ Data Science*
- *Science* 和 *Nature* 中也频繁出现 CSS 论文

## 前沿方向（2025-2026）

1. **AI 与 LLM 驱动的研究范式**：用大语言模型辅助研究设计、
   数据标注、假设生成——**"AI 辅助社会科学"正在成为 CSS 的子领域**

2. **质性+计算方法的融合**：2025 年 IC2S2 主题演讲中，
   Laura Nelson 提出——计算方法不应只是"自动化质性研究"，
   而应以**归纳和溯因推理**为目标，与质性研究的逻辑对齐

3. **应用于政策分析**：COVID-19 建模、气候变化的社会影响、
   公共卫生政策模拟——CSS 正在证明其现实价值

4. **伦理学与方法论反思**：算法偏见、隐私边界、
   数字鸿沟——CSS 的伦理框架建设正在加速

## 批评与挑战

- **"大数据傲慢"**：数据量≠洞察力，相关性≠因果性
- **代表性偏差**：社交媒体用户≠全体人口
- **计算方法≠理论思考**：会跑模型不代表理解社会
- **隐私与伦理**：大规模社会数据收集的边界在哪里？
