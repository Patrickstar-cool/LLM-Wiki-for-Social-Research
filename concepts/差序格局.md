## 概述

**差序格局**是[[费孝通]]在《乡土中国》（1948）中提出的中国社会结构分析概念。
它是理解中国人际关系、社会组织与道德逻辑的**最核心的本土理论工具**。

> "以'己'为中心，像石子一般投入水中，和别人所联系成的社会关系……像水的波纹一般，一圈圈推出去，愈推愈远，也愈推愈薄。"

---

## 核心含义

### "差"与"序"

阎云翔（费孝通研究的重要学者）指出："差序格局"由两个字构成，缺一不可：

| | 差 | 序 |
|---|---|---|
| **维度** | 横向——关系有远近 | 纵向——关系有等级 |
| **逻辑** | 亲疏有别：家人 > 亲戚 > 朋友 > 熟人 > 陌生人 | 尊卑有序：长辈 > 平辈 > 晚辈 |
| **动态** | 可伸缩的同心圆 | 刚性的等级结构 |

**如果只看"差"不看"序"**：差序格局变成了"关系网"的庸俗版本——"谁离我近我帮谁"。
**费孝通说的是更复杂的东西**：与你有关系的人不仅是"近或远"——还有"上或下"。
父亲和老板都在你的波纹里——但波纹的性质不同。

### 与西方"团体格局"的对比

| | 团体格局（西方） | 差序格局（中国） |
|---|---|---|
| **基本单位** | 团体——个体属于团体 | 关系网络——个体是网络的中心 |
| **边界** | 清晰：你是会员/你不是会员 | 模糊：远近渐变，可伸缩 |
| **道德** | 普遍主义——规则适用于所有人 | 特殊主义——因人而异 |
| **公私** | 公私有明确界限 | 公私相对——"家"可大可小 |
| **个体** | 个体权利不可侵犯 | 个体嵌入在关系中——"关系中的自我" |
| **社会观** | 个体先于社会 | 关系先于个体 |

## 日常生活中的差序格局

### 例子一："自己人"和"外人"

你在跟一个人交往的时候——你做的第一件事不是"他是谁"，是**"他跟我是什么关系"**。
一旦确认了关系，一整套**隐含的义务和期望**就被激活：
- 亲戚有困难——你必须帮，不帮是"不认人"
- 朋友托你办事——你不好拒绝——不是怕法律后果，是怕**伤了关系**
- 但对陌生人——你没有相同的义务

这不是"自私"——这是**差序格局下的道德逻辑**：道德不是普遍的——是根据关系远近而有"差等"的。

### 例子二：伸缩的"家"

"我们一家人"——这个"家"可以有多大？

- 和伴侣吵架时："我们家的事你别管"——家 = 核心家庭
- 过年拜祖宗时："全家人都在"——家 = 整个家族
- 在异国遇到同乡："咱们一家人不说两家话"——家 = 同乡网络

**"家"的边界是可伸缩的**——这在西方"团体格局"中是难以想象的。

### 例子三：礼治

熟人社会为什么不需要那么多法律？因为**礼**在起作用：
- 礼不是"成文法"——是**被内化的行为规范**
- 从小在仪式和日常互动中学习——"这样做才对"
- 违反的惩罚不是法律的——是**社会性的**：被指指点点、被孤立、被说"不懂规矩"

## 批评与延伸

### 批评
- **"差序格局"是描述还是规范？**——费孝通是在说中国社会"事实上"是这样，还是说它"应该"是这样？文本中两者交织
- **忽略"尊尊"？**——吴飞指出：差序格局很好地把握了"亲亲"原则（血亲亲近），但对"尊尊"（等级尊卑）的讨论不够
- **适用于现代城市吗？**——在高流动性的城市陌生人社会中，差序格局还有解释力吗？——学者们仍在争论

### 延伸
- 翟学伟提出"**人缘、人情、人伦**"三位一体模式来补充差序格局
- 黄光国区分了**情感性关系、工具性关系、混合性关系**
- **"工具性差序格局"**（李沛良）：在市场经济中，差序格局被工具化——"我帮你不是因为我们有关系——是因为你将来也可能帮我"

## 当代经验验证：Hu (2024) 的全国调查

胡安宁（复旦大学，2024）使用 2022 年"传统文化与认知模式调查"（TCCPS）数据，
对差序格局进行了**首次大规模定量检验**。^[raw/papers/hu-2024-differential-mode-association.md]

### 核心发现

| 发现 | 含义 |
|------|------|
| **差序格局仍然存在** | 中国人的人际信任从家人→亲戚→朋友→熟人→陌生人呈现清晰的波纹状递减。在农村地区更显著。 |
| **亲缘-非亲缘鸿沟** | 对亲属的信任显著高于非亲属——这条边界依然坚固。 |
| **长辈和晚辈比同辈更受信任** | 年轻一代对父母辈的信任高于对兄弟姐妹——"序"的等级维度依然有效。 |
| **配偶信任极高** | 婚姻关系在现代中国社会信任结构中占据了核心位置——这可能是对费孝通时代"家族本位"的修正。 |
| **兄弟姐妹信任随年龄下降** | 部分归因于独生子女政策——没有兄弟姐妹可信任的一代。 |
| **父系原则不再成立！** | 母系和父系亲属受到**同等信任**——这是对传统差序格局的一个重大修正。费孝通时代假设的"父系优先"在现代中国不再普遍成立。 |
| **非亲缘领域三分** | 最信任的密友 > 中等信任的有意义关系者 > 最不信任的陌生人——非亲缘关系本身也是"差序"的。 |

### 理论意义

胡安宁的研究既**证实**了差序格局的持久解释力（波纹结构依然存在），
又**修正**了它的具体形态：
- "差"（亲疏远近）仍然强劲——但亲疏的"边界"在变化（配偶>父母>亲戚的排序可能与费孝通时代不同）
- "序"（等级尊卑）仍然存在——但父系原则的消解表明等级结构本身在民主化
- 差序格局**不是一个静态的传统文化残留**——它在与现代制度（婚姻法、独生子女政策、城市化）互动中不断重塑自身
