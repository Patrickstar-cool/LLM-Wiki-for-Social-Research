## 概述

罗伯特·金·默顿（Robert K. Merton，1910-2003）是帕森斯的学生，
但他不像老师那样建造宏伟的理论宫殿——**默顿用功能论的工具做了让其他社会学家真正能用的东西。**

他是"中层理论"（Middle-Range Theory）的鼓吹者、社会学史上最著名概念的作者之一——
"自我实现的预言""角色模范""焦点群体"——这些概念从默顿的笔下来到日常语言中。

## 生平

- 1910 年出生于费城，东欧犹太移民家庭，原名 Meyer R. Schkolnick
- 少年时变过魔术——后来他说这让他对"人们如何被表面现象欺骗"产生了早期兴趣
- 哈佛大学师从帕森斯——但对老师的宏大理论一直保持距离
- 1941 年加入哥伦比亚大学——在这里待了 40 年，与拉扎斯菲尔德（Lazarsfeld）的量化方法形成了著名的"理论+实证"组合
- 1979 年退休，2003 年去世，享年 92 岁

## 核心贡献

### 1. 中层理论

默顿对社会学理论最持久的方法论贡献：

> 不要追求"解释一切社会现象"的万能理论（像帕森斯那样）。
> **专注于可以被经验检验的、仅限于特定领域的理论。**

- 中层理论不是"低级的"——它是**有边界的、可操作的理论**
- 例子：参照群体理论（"你和谁比决定了你觉得自己是富是穷"）
- 例子：角色冲突理论（"当你同时是母亲和 CEO 时会发生什么"）

**这一定位帮助社会学在"帕森斯的宏大"和"量化实证的零碎"之间找到了立足点。**

### 2. 显性功能与隐性功能

默顿对功能论最重要的修正：

| | 显性功能 | 隐性功能 |
|---|---|---|
| **定义** | 被意图的和被认识到的后果 | 未被意图的和未被认识到的后果 |
| **例子：教育** | 传授知识和技能 | 把 18-22 岁年轻人暂时排除在劳动力市场之外 |
| **例子：祈雨仪式** | 求雨（未必有效） | 增强部落凝聚力（确实有效） |

> **社会学家的工作，是揭示那些未被认识到的隐性功能。**

### 3. 自我实现的预言

默顿最著名的概念（1948）：

> "如果人们将情境定义为真实的，那么它们在后果中就是真实的。"

- 一个谣言说某银行要破产 → 人们蜂拥去取钱 → **银行真的破产了**
- 老师说某个孩子"没有数学天赋" → 孩子内化了这个标签 → **他真的不再努力了**
- 这不是"偏见产生偏见"的同义反复——它揭示了**信念如何通过改变行为而改变现实**

### 4. 越轨的类型学

默顿用**文化目标**和**制度手段**的张力来解释越轨：

| 类型 | 接受文化目标？ | 接受制度手段？ | 例子 |
|------|-------------|-------------|------|
| **顺从** | ✓ | ✓ | "正常"社会成员 |
| **创新** | ✓ | ✗ | 毒贩——想要财富（目标），用非法手段 |
| **仪式主义** | ✗ | ✓ | 官僚——忘记目标，只遵守规则本身 |
| **退却主义** | ✗ | ✗ | 流浪者、隐士 |
| **反叛** | 拒绝现有，追求新目标 | 拒绝现有，追求新手段 | 革命者 |

默顿说：**美国社会过度强调"成功"的目标——但对"合法手段"的提供是不平等的。** 这种张力系统性地制造了越轨。

### 5. 马太效应

> "凡有的，还要加给他，叫他有余；凡没有的，连他所有的，也要夺去。"——《马太福音》

在科学社会学中：已经成名的科学家获得不成比例的声誉——同样的工作，
如果是一个不知名的研究生做的，和一个诺贝尔奖得主做的，受到的**关注完全不同**。

**这不是"不公平"的偶然——它是科学系统的结构性特征。**

## 默顿 vs 帕森斯

| | 帕森斯 | 默顿 |
|---|---|---|
| **理论风格** | 宏大理论——解释一切 | 中层理论——解释特定领域 |
| **对功能论** | 假设一切都有正功能 | 区分正功能/反功能、显性/隐性 |
| **对变迁** | 系统倾向于均衡 | 张力（如目标-手段分裂）是变迁的动力 |
| **遗产** | 理论问题意识 | 可操作的概念和可检验的假设 |

## 批评

- **中层理论足够吗？** 没有宏大理论的统领，"中层"不会变成零碎吗？
- **功能论底色**：尽管修正了帕森斯，"功能"仍然是他分析的核心透镜——冲突论者说他忽视权力
